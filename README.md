📊 Sales Dashboard (Power BI)
📌 Overview
This project presents an interactive Sales Dashboard developed using Microsoft Power BI. The dashboard provides a clear and insightful view of sales performance, helping users analyze key business metrics and make data-driven decisions.

🎯 Objectives
Monitor overall sales performance
Analyze monthly and yearly trends
Compare regional and category-wise sales
Identify top-performing products and segments

📈 Key Features
Dynamic visualizations and filters
Year-over-Year sales comparison
Category and region-wise breakdown
KPI indicators for quick insights
User-friendly and interactive design

📊 Insights You Can Gain
Sales growth trends over time
Best-performing regions and categories
Customer purchasing patterns
Revenue contribution by segments
